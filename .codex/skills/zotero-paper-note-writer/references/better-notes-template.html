PASS 1: 概览 (The Skim Pass)
Tips: 快速浏览（5-10分钟），了解论文大意。阅读标题、摘要、引言、结论以及章节标题。

1. 论文类别 (Category)

这是一篇什么类型的论文？（例如：新算法、系统设计、测量分析、综述...）

2. 核心问题 (Core Problem)

这篇论文试图解决什么关键问题？

3. 主要贡献 (Main Contribution)

作者声称的主要贡献是什么？（用1-2句话总结）

4. 论文结构 (Structure)

快速记录各部分的标题和主要内容，建立一个内容地图。

🔍 关键背景与术语
Tips: 记录在阅读背景 (Introduction/Background) 部分时遇到的不熟悉的概念、缩写或关键的前置知识。

🎓 关键概念

acronyms (缩写)

PASS 2: 精读 (The Detail Pass)
Tips: 仔细阅读全文（约1小时），重点关注图表和方法。目标是理解论文的实现方式和论证过程。

1. 方法 / 模型 / 系统设计 (Method / Model / System Design)

详细描述它是如何工作的。关键组件和流程是什么？（可以截图辅助）

2. 实验设置 (Experiment Setup)

使用了什么数据集？对比了哪些 Baselines？评估指标是什么？

3. 关键图表分析 (Key Figures/Tables Analysis)

（截图并分析）最重要的图表展示了什么？它们如何支持论点？

4. 主要结果 (Main Results)

实验结果是什么？是否达到了作者声称的效果？

5. 疑问点 (Questions)

记录阅读中不理解的术语、概念或论证。

PASS 3: 批判 (The Critical Pass)
Tips: 带着批判性思维重新审视（1-2小时）。目标是评估论文的价值并将其与现有知识联系起来。

1. 优点 / 创新点 (Strengths / Innovations)

这篇论文最大的亮点是什么？哪些部分做得特别好？

2. 缺点 / 局限性 (Weaknesses / Limitations)

作者有哪些未提及的假设？实验是否公平？有哪些局限性？

3. 可改进点 / 未来工作 (Future Work)

你认为这项工作可以如何改进？有哪些值得跟进的未来方向？

4. 相关工作联系 (Connection to Related Work)

这篇论文与你读过的其他论文有什么联系？（证实/反驳/补充了哪些观点？）

5. 个人思考 / 启发 (My Thoughts / Inspiration)

这篇论文对你的研究/工作有什么启发？（你的 "Aha!" 时刻）
